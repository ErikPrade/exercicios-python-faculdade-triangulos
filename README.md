# Classificador de Triângulos em Python 🔺

Este projeto foi desenvolvido como parte da Atividade 1 da faculdade. O objetivo é ler três lados de um triângulo e classificá-lo, utilizando cores no terminal.

## 🚀 Funcionalidades
- Validação matemática (verifica se os lados formam um triângulo).
- Classificação: Equilátero, Isósceles ou Escaleno.
- Interface colorida usando a biblioteca `colorama`.

## 🛠️ Tecnologias
- Python 3.14
- Colorama (Biblioteca de cores)
